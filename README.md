### Команды

- Создай зависимости  
`npm i`  либо `yarn install`

- Режим разработки  
`npm start`  либо `yarn start`


- Command Build Project

- Win 7 x32 portable  
- npm run package -- --platform=win32 --arch=ia32  


- Win 7 x32 installer  
- npm run make  


- Linux  
- npm run package -- --platform=linux --arch=x64
